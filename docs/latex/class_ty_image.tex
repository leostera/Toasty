Loads a resource named name from the resource group that group points to if the current group is different from the one specified, the new group is set to be the current. If the resource name and group equals the current resource name and group then nothing is done. If current resource name differs from pName, current resource group is not pGroup, the object has not yet been built or both the resource name and resource group has not been set previously, set pGroup as current resource group, pName as current resource name and try to load the resource.
