Detailed Description

The almighty clock. returns time in ms from loop to loop and is capable of frame rate capping.

int64 TyClock::Cap ( ) [inline]

Caps.

Returns:
int64 current amount of frames per second or ZERO if frame capping is disabled.

If frame capping is enabled (m_Cap > 0) we will wait long enough to let only m_Cap frames be displayed every second.
