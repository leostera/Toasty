Detailed Description

Actor class to be used within a scene.

**********************************************************************************

Date:
2/08/2011
Version:
0.1
Author:
Leandro Ostera

Actor class to be used within a Scene class that works with states (CTyState) and is meant to be an interface with some virtual methods to be overloaded.
